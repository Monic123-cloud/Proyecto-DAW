## Guía de Comandos Rápidos

```bash
### 1. Clonar el proyecto (Solo la primera vez)

git clone https://github.com/Monic123-cloud/Proyecto-DAW.git


### 2. Creamos nuestra propia rama

git push -u origin Monica

###Guardamos y subimos los avances en nuestra rama

git add .
git commit -m "Explicamos lo que hemos hecho"
git push -u origin TuNombre

###Cuando tengamos algo terminado y comprobado que funciona

####Te pasas a main:
git checkout main

####Te bajas lo que hayan hecho otros:
git pull origin main

####Traes tus cambios de tu rama a main:
git merge Monica

####Subes el main actualizado:
git push origin main

#Actualizamos el projetc de nuestra ficha en Git

## 🛠️ Reglas del Repositorio

Para trabajar en equipo esta tarde sin errores:

1. **Ramas propias**: Cada uno trabaja en su rama (ej: `Monica`).
2. **Actualización**: Haz `git pull origin main` antes de empezar.
3. **Mensajes claros**: Pon nombres descriptivos a tus commits.

## 📂 Estructura de Carpetas

- `/src` - Código fuente.
- `/docs` - Documentación del proyecto.
- `/sql` - Scripts de la base de datos.
```
